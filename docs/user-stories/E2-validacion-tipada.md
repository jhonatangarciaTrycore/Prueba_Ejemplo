# E2 — Validación tipada (opcional)

**Como** desarrollador
**Quiero** usar validación tipada (p. ej. Pydantic, Joi, Zod, Bean Validation)
**Para** delegar validaciones y obtener mensajes de error consistentes.

## Requisitos
- Reemplazar validación manual por la biblioteca idiomática del stack.
- Documentar el esquema en el README o en OpenAPI.

## Criterios de aceptación
- Validaciones usan el mecanismo idiomático seleccionado.
