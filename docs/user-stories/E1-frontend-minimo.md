# E1 — Frontend mínimo (opcional)

**Como** usuario no técnico
**Quiero** una página web simple que consuma `POST /match`
**Para** poder probar el endpoint sin usar curl o Postman.

## Requisitos
- Formulario para pegar/introducir JSON de request.
- Botón "Enviar" que muestre la respuesta en formato legible.

## Criterios de aceptación
- Página estática accesible localmente y documentada en README.
